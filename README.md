# microservice

### POST JSON Parameter
| Parameter | Type | Describe |
| ------ | ------ | ------ |
| Req | string | 可以隨意輸入當回應值 |
| Http_fail_rate | int | 0~100 |
| Http_Status_code | int | 可用來當http status code |
| Http_Delay_rate | int | 0~100 |
| Http_Delay | int | ms |
| Next | string | 要去的下一個網址可不填寫 |
